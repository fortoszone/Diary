## Tech Stack
* MongoDB - Authentication, Realm, Device Sync, Atlas, Kotlin SDK
* Room Database
* Firebase - Authentication, Storage
* Google Authentication 
* Splashscreen API
* JWT Authentication
* Dependency injection with Hilt
* MVVM Design Pattern
* UI in Compose
* Material3
* Kotlin Coroutines and Flow
* Animation with Lottie
